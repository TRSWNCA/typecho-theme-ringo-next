# Typecho Theme Ringo

一个朴素的 Typecho 主题，灵感及配色来自于 Mikato 的 Journal 主题。自带 Highlight.js 并有配套的 KaTeX 插件支持。

**这并不是一个正式发布的版本，很多功能和设置都在开发的过程中。**
