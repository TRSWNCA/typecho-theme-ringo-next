# Typecho Theme Ringo

一个朴素的 Typecho 主题，灵感及配色来自于 Mikato 的 Journal 主题。自带 Highlight.js 并有配套的 LaTeX 插件支持。

### Demo

可以直接前往 [memset0.cn](//memset0.cn) 查看效果。
